# Design Assets

The images in `docs/` are handcrafted SVG files created directly in code.

## Why SVG

- Crisp quality on all screens
- Small file size
- Easy to edit in text
- Perfect for GitHub README visuals

## Files

- `docs/banner.svg` - top hero banner
- `docs/preview-ui.svg` - UI mock preview
- `docs/termux-phone.svg` - phone + Termux setup visual
- `docs/repo-showcase.svg` - feature showcase panel

## Editing Tips

- Change colors in gradient blocks first
- Keep text short for readability
- Use rounded corners and subtle shadows
- Prefer consistent spacing for a premium look
