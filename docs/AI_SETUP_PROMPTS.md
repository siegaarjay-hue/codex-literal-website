# AI Setup Prompts

Use one prompt below with your preferred AI assistant.

## Universal Prompt

```text
Help me run the GitHub repo "codex-literal-website" locally.
I am a beginner.
My operating system is: <Windows / macOS / Linux>.
Give exact commands only, one step at a time, with expected output checks.
Include how to:
- install or verify Node.js 20+
- run npm install
- start the app
- open it in browser
- stop it cleanly
- troubleshoot common errors
```

## Windows Prompt

```text
I am on Windows 11 and I need beginner-friendly steps to run "codex-literal-website".
Use PowerShell commands only.
Assume I have never used npm before.
Add fallback steps if execution policy or PATH causes issues.
```

## macOS Prompt

```text
I am on macOS and I need beginner-friendly steps to run "codex-literal-website".
Use zsh/bash commands only.
Check Node.js version first and include Homebrew install commands if needed.
```

## Linux Prompt

```text
I am on Linux and I need beginner-friendly steps to run "codex-literal-website".
Use shell commands only.
Include Node.js install options for apt and dnf if Node is missing.
```
