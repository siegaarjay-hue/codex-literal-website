# Setup Guide (Beginner Friendly)

This guide gives copy-paste commands by operating system.

## Windows (PowerShell)

```powershell
cd path\to\codex-literal-website
node -v
npm -v
npm install
npm run start
```

Open: `http://127.0.0.1:8000/`

Stop:

```powershell
npm run stop
```
